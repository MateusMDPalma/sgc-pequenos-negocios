# Sistema de Gestão Comercial para Pequenos Negócios

Projeto acadêmico da disciplina de Análise e Desenvolvimento de Sistemas.

## Descrição
Sistema web para gestão de clientes, produtos, vendas e relatórios de pequenos negócios.

## Objetivo
Desenvolver um sistema com autenticação, cadastro de clientes, cadastro de produtos, registro de vendas, controle de estoque e relatórios.

## Tecnologias previstas
- Python
- Django
- Django REST Framework
- PostgreSQL
- JWT
- GitHub

## Estrutura inicial
- docs/: documentação da entrega 1
- sql/: script de criação do banco
- src/: código-fonte do sistema
